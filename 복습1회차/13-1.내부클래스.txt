클래스 내부에서 정의된 클래스

1. 인스턴스 내부 클래스 - 멤버 변수
							JDK - 16 이전, 이후 
							 
	외부 클래스 생성 -> 내부 클래스 생성 
	
2. 정적 내부 클래스 - 정적 변수
	외부 클래스 생성 상관없이 내부 클래스 생성
	정적 내부클래스 -> 외부 클래스가 데이터 영역 올라갔을때 초기화
	-> 정적 내부클래스는 정적 메서드와 성질이 유사
	-> 내부에서 외부 인스턴스 메서드 X, 외부 멤버 변수 X 
	-> 외부 정적 변수, 정적 메서드 -> 접근 가능 
	
	

3. 지역 내부 클래스 - 인스턴스 메서드
	메서드 내부 -> 지역 -> 메서드가 호출되면 실행되는 지역(스택)
	
	인터페이스, 추상클래스 -> 인스턴스 생성 불가(구현 부분이 구현되지 않았기 때문에)
	예외로 인스턴스 만들수 있는 조건 
	1) 지역 내부
	2) 클래스의 변수 
	 -> 미구현된 메서드 -> 전부 구현
