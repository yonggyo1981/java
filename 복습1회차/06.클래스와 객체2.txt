3. static 변수(클래스 변수)

static 자료형

static - 정적  -> 그대로 머물러 있다, 동적 메모리로 이동 X 

정적 메모리  - 데이터 영역 
 (한번 올라가면 그상태 그대로 유지 )
 
 
 
동적 메모리 - 스택, 힙영역

static -> 메서드 

static 변수(1) --> 멤버 변수(힙에 인스턴스로 생성)(2)


4. 변수 유효범위
	정적변수(static 변수)
		프로그램이 로딩될때 -> 데이터 영역(인스턴스 생성 X)
		정적 메서드, 인스턴스 메서드 내에서도 접근 가능
	
	멤버변수 - 클래스 내부에 정의 
		힙에 인스턴스로 생성되었을때 접근 가능
	
	지역변수
		메서드 내부에서 정의 
		스택 메모리 영역에 올라갈때 활성화
		(메서드가 실행되는 동안에만 활성화)
		-> 실행이 종료가 되면 -> 스택에서 제거
		
		메서드 지역 내에서만 존재하는 변수
