1. 스트림이란?
여러 자료의 처리에 대한 기능을 구현해 놓은 클래스

2. 스트림의 특징(P478)
	1) 자료의 대상과 관계 없이 동일한 연산을 수행한다
	   정수, 문자, 다양한 클래스 배열, 컬럭션 프레임워크 ... 
	   -> 스트림으로 변환 -> 동일한 메서드 사용 
	   forEach, map, filter ..... 
	2) 한 번 생성하고 사용한 스트림은 재사용 불가
	
	3) 스트림 연산은 기존 자료를 변경하지 않는다
	
	4) 스트림 연산은 중간 연산과 최종 연산 
	중간 연간 -> 데이터를 가공 -> 다음 연산으로 넘긴다(filter, map, sorted)
	최종 연산 -> 스트림 소비 -> 최종적으로 결과 도출(출력, 반환 ...)
					(forEach, count(), sum() ...)
