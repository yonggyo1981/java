입출력(입력, 출력)
IO -> Input, Output
스트림 -> 데이터가 이동하는 가상의 통로

단방향 스트림
	- 입력 스트림
	
	- 출력 스트림
	
	
입력스트림
	- InputStream - 추상클래스 (바이트 단위의 입력 스트림)
			- 바이트 단위의 입력스트림이 따라야할 규칙, 규약 - 추상 메서드  정의하는 클래스
			FileInputStream 
			FilterInputStream
			BufferedInputStream
			DataInputStream 
			ObjectInputStream 
			
	- Reader - 추상클래스(문자 단위의 입력 스트림)
			- 문자 단위의 입력 스트림이 따라야할 규칙, 규약 - 추상 메서드로 정의하는 클래스
			FileReader
			BufferedReader 
			inputStreamReader 

출력스트림
	- OutputStream - 추상 클래스 (바이트 단위의 출력 스트림)
			- 바이트 단위의 출력스트림이 따라야할 규칙, 규약 - 추상 메서드로 정의하는 클래스
			FileOutputStream 
			BufferedOutputStream
			DataOutputStream
			ObjectOutputStream
			FilterOutputStream
			
	- Writer - 추상 클래스 (문자 단위의 출력 스트림)
			- 문자 단위의 출력 스트림이 따라야할 규칙, 규약 - 추상메서드로 정의하는 클래스
			FileWriter,
			BufferedWriter 
			OutputStreamWriter 
	
	
기반스트림
	- 직접 읽고 쓰는 스트림
	FileInputStream, FileOutputStream, 
	FileReader, FileWriter
	(생성자 인수 -> 파일, 콘솔)
	File file, String fileName ....
