13.1 내부 클래스 
	1. 인스턴스 내부 클래스 
		JDK 16 이상 -> 정적 변수, 정적 메서드 정의 가능 
						
		JDK 15 이하 -> 정적변수, 정적 메서드 정의 불가능
						-> 내부 클래스의 인스턴스는 외부 클래스가 인스턴스로 생성된 후 
		
	2. 정적 내부 클래스
		일반 클래스와 큰 차이가 없다, 외부 클래스의 인스턴스 생성과 상관 없이 인스턴스 생성 가능 
	
	3. 지역 내부 클래스
		메서드 안에 정의된 클래스 
		
		인터페이스, 추상 클래스 -> 미구현된 메서드가 있으므로 인스턴스 생성 X
		
		-> 인스턴스가 만들어질 수 있는 상황 
		    -> 메서드 안에서(지역 내부)
			-> 클래스의 멤버 변수에서
			+ 미구현된 메서드를 구현 함으로써 
		
	
	4. 익명 내부 클래스

13.2 람다식 
	-> 인터페이스 + 지역 내부 클래스를 이용해서 구현 
	-> 정의 되어 있는 메서드는 반드시 1개
