내부 클래스란?
	클래스 내부에 클래스가 정의 되어 있는 것 

1. 인스턴스 내부 클래스 - 멤버변수(인스턴스 변수, 인스턴스 메서드)

jdk15 이하 - 내부에서 static 변수, 메서드 X
			   -> 내부클래스의 인스턴스 생성을 하려면 외부 클래스의 인스턴를 먼저 생성 
	
jdk16 이상 - static 변수, 메서드 
			   -  데이터 영역에 내부 클래스더라도 클래스 파일이 올라갈때 처리
			   

2. 정적 내부 클래스 - 정적 변수, 정적 메서드
	- 외부 클래스의 인스턴스 생성과 관련없이 독자적으로 생성 가능
	- 일반 외부 클래스와 거의 동일(외부 클래스를 통해서 접근)


3. 지역 내부 클래스 - 메서드 안에 정의되어 있는 클래스



4. 익명 내부 클래스
